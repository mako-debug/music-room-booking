# 新米蘭音樂教室 — 琴房預約系統 操作手冊

## 登入

1. 用手機掃描教室的 QR Code，或在瀏覽器輸入網址
2. 輸入 Email 和密碼，點「登入」
3. 建議使用 Chrome 或 Safari 瀏覽器（不要用 LINE 內建瀏覽器，否則每次要重新登入）
4. 第一次登入後瀏覽器會問「要儲存密碼嗎？」，選儲存，之後就不用再輸入

## 查看課表

### 日檢視（手機預設）
- 一次看到 7 間教室當天的排課狀況
- 有色塊的格子 = 已預約（顯示老師名和學生名）
- 點上方的 `<` `>` 切換日期

### 週檢視（電腦預設）
- 一次看到一間教室整週的排課
- 上方下拉選單可切換教室（1號~7號）
- 今天的那一欄會有藍色底色標示
- 點 `<` `>` 切換上下週

### 切換檢視
- 左上角下拉選單可以切換「日」和「週」
- 系統會記住你的選擇，下次打開不用重選

## 預約教室（老師/管理員）

1. 在課表上點一個**空白格子**
2. 彈出「新增預約」視窗，教室/日期/時間已自動帶入
3. 選擇**時長**（30分鐘 / 1小時 / 1.5小時 / 2小時）
4. 填入**學生姓名**（必填）
5. 填入**用途**（選填，例如「鋼琴課」）
6. 如果是固定課，勾選**每週重複**，選擇重複幾週（2~4 週）
7. 點「確認預約」

### 注意事項
- 只能預約未來 1 個月內的時段
- 同一教室同一時段不能重複預約（會提示衝突）
- 結束時間不能超過 22:00
- 管理員可以從「老師」下拉選單代其他老師預約

## 查看/修改/取消預約

1. 點課表上已預約的**色塊**
2. 彈出「預約詳情」視窗，顯示完整資訊

### 修改
- 點「修改」→ 學生姓名和用途變成可編輯 → 改完按「儲存」
- 只有管理員或預約的老師本人可以修改

### 取消
- 點「取消預約」→ 確認後取消
- 如果是重複排課，可以選擇：
  - 「只取消這次」— 只取消這一堂
  - 「取消此週之後全部」— 取消這堂及之後所有同組的預約
- 只有管理員或預約的老師本人可以取消

## 修改自己的姓名

1. 點右上角的 **👤 你的名字 ✏️**
2. 輸入新名字 → 點「存」
3. 所有已預約的課表會同步更新新名字

## 修改自己的密碼

1. 點右上角的 **🔑** 圖示
2. 輸入目前密碼
3. 輸入新密碼（至少 6 個字元）
4. 再輸入一次確認
5. 點「確認更新」

## 帳號管理（僅管理員）

點右上角的「帳號管理」進入管理頁面。

### 新增帳號
1. 填入姓名、Email、密碼（至少 6 字元）
2. 選擇角色：老師（可預約）或 學生（僅查看）
3. 點「建立帳號」

### 修改帳號
- 點「修改」→ 可改姓名、Email、角色、色系
- 色系：點色圓點選擇該老師在課表上的顯示顏色
- Email 修改會同時更新登入帳號

### 重設密碼
- 點「重設密碼」→ 輸入新密碼 → 確認更新
- 適用於老師忘記密碼的情況

### 停用/啟用帳號
- 點「停用」→ 該帳號無法登入，但已預約的課表會保留在課表上
- 停用後可以點「啟用」恢復
- 適用於老師離職的情況，不需要刪除帳號

### 刪除帳號
- 點「刪除」→ 確認後刪除
- 已預約的課表會保留（不會被刪除）
- 建議用「停用」代替刪除

## 常見問題

**Q: 每次打開都要重新登入？**
A: 用 Chrome 或 Safari 開網址，不要用 LINE/Facebook 內建瀏覽器。登入一次後瀏覽器會記住。

**Q: 手機上教室太多看不完？**
A: 日檢視可以左右滑動。或切換到週檢視，一次只看一間教室。

**Q: 忘記密碼怎麼辦？**
A: 請管理員到帳號管理幫你重設密碼。

**Q: 預約時顯示「此時段已有預約」？**
A: 該教室的這個時段已經被其他老師預約了，請選其他時段或其他教室。

**Q: 兩個老師名字一樣怎麼分辨？**
A: 每位老師會自動分配不同顏色，管理員也可以在帳號管理指定色系。
